# kitra

## Setup
Clone the repo and install the dependencies.

```bash
git clone git@github.com:eemsquared/kitra.git
```
```bash
npm install
```

To start the express server, run the following

```bash
node index.js
```

Open [http://localhost:3306](http://localhost:3306)

## Endpoints

`GET /treasures`
`POST /users`
